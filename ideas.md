# Siga o Dinheiro - Design Ideas

## Estilo Escolhido: Editorial Financeiro Sofisticado

### Design Movement
Inspirado em publicações financeiras premium como The Economist, Financial Times e Bloomberg. Tipografia editorial forte com hierarquia clara.

### Core Principles
1. **Legibilidade Suprema** - Foco total na leitura confortável de artigos longos
2. **Hierarquia Visual Clara** - Títulos impactantes, corpo de texto confortável
3. **Credibilidade** - Cores sóbrias que transmitem seriedade e confiança
4. **Navegação Intuitiva** - Acesso rápido a todos os artigos

### Color Philosophy
- **Fundo Principal**: Preto profundo (#0a0a0a) - transmite sofisticação
- **Texto Principal**: Branco suave (#f5f5f5) - alta legibilidade
- **Destaque/Accent**: Dourado (#d4af37) - riqueza, finanças, poder
- **Secundário**: Cinza médio (#888888) - informações complementares

### Layout Paradigm
- Sidebar fixa à esquerda com navegação de artigos
- Área de leitura ampla e centralizada
- Tipografia grande para conforto de leitura

### Signature Elements
1. Linha dourada como separador visual
2. Cards de artigos com hover sutil
3. Ícones minimalistas para categorias

### Typography System
- **Títulos**: Playfair Display (serif elegante)
- **Corpo**: Inter (sans-serif legível)
- **Tamanhos**: Títulos grandes (2.5rem+), corpo confortável (1.125rem)

### Animation
- Transições suaves de 200-300ms
- Hover states com elevação sutil
- Fade-in para conteúdo carregado
